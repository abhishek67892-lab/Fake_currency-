# Fake_currency-
## 💰 Fake Currency Detection System

This project is a machine learning-based system designed to detect counterfeit currency notes. It analyzes various features of a currency note such as texture, patterns, and security elements to determine authenticity.

### 🚀 Features
- Detects fake vs real currency notes
- Uses machine learning algorithms for classification
- User-friendly interface for image input
- Fast and accurate predictions
